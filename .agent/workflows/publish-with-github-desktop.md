---
description: How to publish your repository using GitHub Desktop
---

1. **Open GitHub Desktop**.

2. **Add Local Repository**:
   - Go to **File** > **Add Local Repository...** (or press `Ctrl + O`).
   - Click **Choose...** and navigate to your project folder:
     `c:\Users\adero\Documents\My Code\threat-intel-app`
   - Click **Add Repository**.

3. **Publish**:
   - You should see your current changes (or "Initial commit").
   - Click the **Publish repository** button in the top toolbar.
   - **Name**: `threat-intel-app`
   - **Description**: (Optional) "Cybersecurity Threat Intelligence Dashboard"
   - **Organization**: None (to publish to your personal account).
   - Uncheck "Keep this code private" if you want it to be public.
   - Click **Publish Repository**.

4. **Verify**:
   - Once finished, click **View on GitHub** to see your live repository.
